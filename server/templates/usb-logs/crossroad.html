Available endpoints are accessible from selectBox above

Endpoints are:

URL Purpose
/logs-web Shows all saves Keyman logs. User can filter through license, team and user
/ldlogs-web Shows all saves LD debugger logs. User can filter through license, team and user
/devices-web Shows all Keyman devices saved in database and its last state (connected, disconnected)
/body-devices-web Shows all LD Body devices saved in database and its last state (connected, disconnected)
/teams-web Shows all teams currently saved in database and its members
/pcs-web Shows all PCS currently saved in database and its team
/licenses-web Shows all Licenses currently saved in database and its expiration date
